# 📊 Customer Behavior Analysis

## 📌 Project Overview
Customer Behavior Analysis is a data analytics project focused on understanding customer purchasing patterns, product preferences, and sales trends. The project uses SQL, Python, and Power BI to transform raw customer transaction data into actionable business insights.

## 🎯 Business Problem
Businesses often struggle to understand customer buying behavior, identify top-performing products, and optimize sales strategies. This project aims to uncover meaningful patterns from customer data to support data-driven decision-making.

---

## 🛠️ Tech Stack

- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL
- Power BI
- Microsoft Excel

## 🔍 Project Workflow

### 1️⃣ Data Collection
- Imported customer transaction dataset.
- Examined data quality and structure.

### 2️⃣ Data Cleaning
- Removed duplicates.
- Handled missing values.
- Standardized column formats.

### 3️⃣ Exploratory Data Analysis (EDA)
- Customer purchase behavior analysis.
- Category-wise sales analysis.
- Product popularity analysis.
- Customer segmentation insights.

### 4️⃣ SQL Analysis
- Top-selling products.
- Most purchased categories.
- Customer purchase frequency.
- Revenue-based analysis.

### 5️⃣ Dashboard Development
- Interactive Power BI dashboard.
- KPI tracking.
- Sales trends visualization.
- Customer insights reporting.
